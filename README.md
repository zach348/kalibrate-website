# Kalibrate Website

Professional website for Kalibrate - Interactive meditation and focus training app.

## Overview

This is a static website built for GitHub Pages, replacing the previous Wix site with:
- Faster loading times
- Better mobile responsiveness  
- Professional design using Kalibrate's brand colors
- App screenshots showcasing the session flow

## Brand Colors

- **Mint**: #77FDC7 (primary accent)
- **Gray**: #A09EA1 (secondary text)
- **Dark**: #242424 (primary text/backgrounds)

## Structure

- `index.html` - Landing page with hero, how-it-works section, and CTA
- `privacy-policy.html` - Privacy policy (App Store compliance)
- `terms-of-use.html` - Terms of use (App Store compliance)
- `css/style.css` - All styling
- `images/` - App screenshots and assets

## Legal URL Continuity

The privacy policy and terms of use maintain the same URL paths as the previous site:
- `/privacy-policy` → `/privacy-policy.html`
- `/terms-of-use` → `/terms-of-use.html`

This ensures App Store compliance without needing to update any references.

## GitHub Pages Setup

1. Create repository
2. Upload files
3. Enable GitHub Pages in repository settings
4. Configure custom domain to point kalibrate.me to GitHub Pages
5. Update DNS records to point to GitHub Pages servers

## Deployment

The site is ready to deploy - all files are self-contained with no build process required.
